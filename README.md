- 🤖 Self-improving AI that learns from every task
- 💬 123+ AI Providers support (OpenAI, Claude, Gemini, Deepseek, Grok...)

- ⏰ Automation & Scheduling (Cron jobs, daily reports)
- 🔌 Plugin system with 40+ built-in tools
